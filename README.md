# Grafana Dashboard: _Spring Boot Executors_

> To use this dashboard, import it from
> [the official website](https://grafana.com/grafana/dashboards/20806-spring-boot-executors)
> or use
> [dashboard.json](dashboard.json).

[Grafana Dashboard](https://grafana.com/docs/grafana/latest/dashboards)
of
[Spring Boot](https://spring.io/projects/spring-boot)
(by
[Micrometer](https://micrometer.io)
metrics from
[Prometheus](https://prometheus.io))
for executors.

[About metrics](https://github.com/micrometer-metrics/micrometer/blob/main/micrometer-core/src/main/java/io/micrometer/core/instrument/binder/jvm/ExecutorServiceMetrics.java#L334).

Supported Spring Boot `3.x`.

See details in [GitHub](https://github.com/alexengrig/grafana-dashboard-spring-boot-executors).
